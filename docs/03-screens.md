# 03. 화면 구조

Next.js App Router 기준. `(dashboard)` 그룹은 로그인한 상담원만 접근.

## 라우트 맵

| 경로 | 화면 | 설명 |
|------|------|------|
| `/login` | 로그인 | Supabase Auth |
| `/` | 대시보드 | 오늘 상담 현황, 미처리 건수, 유형별 통계 |
| `/conversations` | 상담 목록 | 필터(상태·유형·거래처·담당), 검색, 정렬 |
| `/conversations/[id]` | 상담 상세 | 메시지 스레드 + AI 답변 패널 + 발송 |
| `/faqs` | FAQ 목록/관리 | 등록·수정·삭제, 유형별 |
| `/categories` | 상담유형 관리 | 대/소분류 트리 편집 |
| `/clients` | 거래처 목록 | 거래처 CRUD |
| `/clients/[id]` | 거래처 상세 | 해당 거래처 상담이력 집계 |
| `/search` | 상담이력 검색 | 키워드 + 의미 검색 결과 |
| `/settings` | 설정 | 연동 키, 상담원 계정(admin) |

## 핵심 화면 상세

### 상담 상세 `/conversations/[id]` — 가장 중요
좌: 메시지 스레드 / 우: 컨텍스트 패널.

```
┌───────────────────────────────┬──────────────────────────┐
│  고객명 · 거래처 · [유형 뱃지]   │  거래처 정보              │
│  ───────────────────────────  │  최근 상담 이력           │
│  고객: 배송이 안 와요            │  ──────────────────────  │
│  고객: 송장번호 1234            │  ▣ AI 답변 초안           │
│       ...                     │  "확인 결과 ..."          │
│  상담원: 확인하겠습니다           │  [참고 FAQ 3건]          │
│  ───────────────────────────  │  [채택] [수정] [폐기]     │
│  [입력창...............] [발송]  │                          │
└───────────────────────────────┴──────────────────────────┘
```

동작:
- 새 고객 메시지 수신 시 자동으로 AI 초안 생성(`ai_responses` draft).
- 상담원: **채택**(그대로 발송) / **수정 후 발송**(edited_answer 저장) / **폐기**.
- 발송 시 `/api/kakao/send` 호출 → `messages`(agent) 추가.
- 유형 뱃지 클릭 시 수동 재분류 가능.

### 상담 목록 `/conversations`
- 필터: 상태(open/pending/closed), 유형, 거래처, 담당 상담원, 기간.
- 정렬: 최근 메시지순(기본).
- 행: 고객명 · 마지막 메시지 미리보기 · 유형 · 미처리 표시 · 시각.
- 실시간: Supabase Realtime 구독으로 신규 메시지 뱃지 갱신(확장).

### FAQ 관리 `/faqs`
- 목록(유형 필터) + 등록/수정 폼.
- 저장 시 `answer`/`question` 임베딩 자동 생성([05](05-ai-pipeline.md)).
- 활성/비활성 토글.

### 상담이력 검색 `/search`
- 검색창 1개 → 키워드(pg_trgm) + 의미(pgvector) 하이브리드 결과.
- 결과: 매칭 메시지 스니펫 → 클릭 시 해당 상담 상세로 이동.
- 필터: 기간·유형·거래처.

### 거래처 상세 `/clients/[id]`
- 거래처 기본정보 + 상담 건수/유형 분포 + 상담 목록.

## 컴포넌트(공용)

- `AiDraftPanel`(채택/수정/폐기)
- `CategoryBadge`, `CategoryTreeEditor`
- `FaqForm`, `FaqList`
- `SearchBar`, `SearchResults`
- `ClientPicker`

## 권한별 노출

- **agent**: 상담/FAQ/검색/거래처 조회.
